# Subsection Draft Template

Use one file per subsection:

- path: `subsection_drafts/<section-id>.md`
- example: `subsection_drafts/5.1.2.md`

````markdown
# <section-id> <subsection title>

## Goal
- What this subsection must establish in the full report.

## Grounded Draft

Write complete subsection prose here.
Use source handles such as `[@source-id]` inline.

Minimum standard for analytical subsections:

- at least 2 prose paragraphs
- each paragraph at least 3 sentences (recommended 5+)
- at least 2 unique citations (recommended 3+ for core subsections)
- include caveat/boundary discussion, not only positive claims

## Self-check
- evidence_coverage: high | medium | low
- overclaim_risk: high | medium | low
- needs_retrieval_reopen: yes | no
- paragraph_count:
- min_sentence_count_per_paragraph:
- unique_citation_count:
- notes:

## Merge Notes
- Which parent section this subsection belongs to.
- Any transition hints for neighboring subsections.
````
