# Template Policy

Use the bundled template only for generic academic drafts.

The default template should:

- compile on common TeX distributions
- work with `natbib` and `bibtex`
- support `\section`, `\subsection`, and `\subsubsection`
- stay readable and editable

Do not claim that the default template is venue-compliant.

When the user provides a template:

- preserve the template structure
- inject generated body content into the expected body slot
- avoid rewriting venue macros unless necessary
