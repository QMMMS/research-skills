# Source Reliability Policy (Template)

Use this file to keep source filtering explicit and auditable.

## Tier A: Preferred

- peer-reviewed venues
- official organization docs
- canonical preprint servers

## Tier B: Caution

- blog posts by domain experts
- project documentation without archival references

Rule:
- keep only if corroborated by Tier A evidence

## Tier C: Exclude

- anonymous forum posts as primary evidence
- unverifiable claims without citations
- generated summaries without source trace

Rule:
- do not use as sole support for analytical claims
